1. 二分查找

```
int left = 0;
int right = array.size()-1;
for(;left < right-1;)
{
  int mid = (left + right)/2;
  if(array[mid] == target)
    return true;
  else if(array[mid] > target)
    right = mid;
  else // array[mid] < target
    left = mid;
}
// left == right-1
if( !array.empty() && (array[left] == target || array[right] == target))
  return true;
```
