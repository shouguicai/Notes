1. 二分查找

```
int left = 0;
int right = array.size()-1;
for(;left <= right;)              // <= 
{
  int mid = left + (right - left)/2; // 防止加法溢出
  if(array[mid] == target)
    return true;
  else if(array[mid] > target)
    right = mid - 1;                // mid - 1
  else // array[mid] < target
    left = mid + 1;                 // mid + 1
}
```
